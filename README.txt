This Add-on seamlessly integrates PayPal with Contact Form 7.

== Description ==

Each contact form can have its own PayPal settings. When a contact form is integrated with PayPal, and the user submits the form it will send an email also & it will auto redirect to PayPal.

If you have any problems, questions, or issues about this PayPal plugins then please create a support request and we will get back to you as quickly as possible.

The administrator can see all the forms data listing with PayPal transaction details.

Note: This PayPal plugin works with both the old and new Contact Form 7 interface. A PayPal account is required to use this plugin.

== Installation ==

This section describes how to install the plugin and get it working.

Steps:-

1. Sign in to your WordPress site as an administrator.
2. In the main menu go to Plugins -> Add New.
3. Search for "Pay with Contact Form 7" Add-on and click on the install button or Download and Upload `pay-with-contact-form-7.zip` to the `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Go into Contact->Paypal Settings, and configure the PayPal settings.
6. From Contact menu you can choose any form, where you will get PayPal activation option. Admin side you can find Contact->Any created Form->PayPal Tab Where you can 
enable PayPal for that form. There is also options to add Item Description, Item Price and Item ID/SKU.

== Frequently Asked Questions ==

= Does it require Plugin? =

It is connected with Contact Form 7 so the user has to install contact form 7 plugin before adding this addon.

= How it works? =

After activating this plugin, Need to fill required fields from Contact->paypal Settings section, and after this go back to created Form and check enable PayPal option from Paypal tab.

= Can this Plugin Display all Form's details in Admin side? =

Yes, This plugin Display all forms details in admin side with PayPal transaction details.
